# 📚 PKMS — Personal Knowledge Management System

**PKMS** — это система для сбора и упорядочивания знаний о себе и о своем вкладе в этот мир. Проект состоит из трех основных компонентов: мобильное приложение для чтения, веб-интерфейс для управления и backend-сервер для синхронизации данных.

---

## 📱 Компоненты проекта

### 1. Mobile — PKMS Reader

Мобильное приложение для чтения электронных книг с отслеживанием статистики.

| Форматы | EPUB, PDF, Audiobook, CBZ, DiViNa |
|---------|-----------------------------------|
| **Статистика** | Страницы, время чтения, прогресс по дням |
| **Синхронизация** | Отправка данных на сервер |
| **TTS** | Озвучивание книг |

**Технологии:** Kotlin, Jetpack Compose, Readium SDK, Room, Retrofit

**[→ Подробнее о Mobile](https://github.com/tganeev/pkms-reader/blob/main/README.md)**

---

### 2. Backend — PKMS API

Серверная часть для хранения и синхронизации данных.

| Функция | Описание |
|---------|----------|
| **REST API** | Эндпоинты для синхронизации книг и статистики |
| **База данных** | PostgreSQL с миграциями через Flyway |
| **Аутентификация** | JWT (в разработке) |
| **CORS** | Настроен для взаимодействия с фронтендом |

**Технологии:** Spring Boot 3, Kotlin, PostgreSQL, JPA/Hibernate

**[→ Подробнее о Backend](backend/README.md)**

---

### 3. Web — PKMS Dashboard

Веб-интерфейс для управления библиотекой и просмотра статистики.

| Функция | Описание |
|---------|----------|
| **Календарь** | Отслеживание практик по дням |
| **Стандарты** | Управление нормативными значениями |
| **Библиотека** | Просмотр и редактирование книг |
| **Статистика** | Графики прогресса |

**Технологии:** React, JavaScript, CSS Modules

**[→ Подробнее о Web](frontend/README.md)**
